2x3 Matrix @module mat2d

@description A mat2d contains six elements defined as: 

[a, c, tx,
 b, d, ty]

This is a short form for the 3x3 matrix: 

[a, c, tx,
 b, d, ty,
 0, 0, 1]

The last row is ignored so the array is shorter and operations are faster.

{Array}a the array of vectors to iterate over
{Number}stride Number of elements between the start of each vec2. If 0 assumes tightly packed
{Number}offset Number of elements to skip at the beginning of the array
{Number}count Number of vec2s to iterate over. If 0 iterates over entire array
{Function}fn Function to call for each vector in the array
{Object}[arg] additional argument to pass to fn
Returns
{Array} a @function
